# Shaler Area Wrestling Website

A simple, clean website for the Shaler Area Wrestling team.

## Setup on GitHub Pages

1. Create a new repository on GitHub
2. Upload these files:
   - index.html
   - styles.css
   - README.md

3. Go to Settings â†’ Pages
4. Select "main" branch as source
5. Your site will be live at: `https://yourusername.github.io/repository-name/`

## Making Updates

Just edit `index.html` directly on GitHub:
1. Click on the file
2. Click the pencil icon to edit
3. Make your changes
4. Click "Commit changes"

## What to Update

- **Links**: Change the Dropbox and registration URLs as needed
- **Text**: Update season info, team descriptions, etc.
- **Sponsors**: Update the list at the bottom
- **Contact**: Change email and social media links

That's it!
